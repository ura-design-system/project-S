## Details of color palette.
If you are interested how the above colors are generated, below shows the code. Generally, tint and shades techniques were used. However, the tint was also combined with a bit of "lighten" to produce brigher hues for the lighter colors throughout the whole palette
```js
/// Slightly lighten/darken a color
@function tint-color($color, $percentage) {
    @return mix(white, lighten($color, $percentage/3), $percentage);
}

@function shade-color($color, $percentage) {
  @return mix(black, $color, $percentage);
}

// URA Color palatte

$blue: #006ae4 !default; //primary //link
$indigo: #4f46e5 !default; //secondary 
$purple: #9333ea !default; 
$pink: #db2777 !default; 
$red: #DA291C !default; // URA Brand Color
$orange: #ea580c !default; 
$yellow: #ca8a04 !default; 
$green: #15803d !default;  
$teal: #0d9488 !default; 
$cyan: #0891b2 !default; 
$gray: #344054 !default; 

//e.g of the tint and shades rules for the blue palette
$blue-100: tint-color($blue, 90%) !default;
$blue-200: tint-color($blue, 80%) !default;
$blue-300: tint-color($blue, 60%) !default;
$blue-400: tint-color($blue, 40%) !default;
$blue-500: tint-color($blue, 20%) !default;
$blue-600: $blue !default;
$blue-700: shade-color($blue, 20%) !default;
$blue-800: shade-color($blue, 40%) !default;
$blue-900: shade-color($blue, 60%) !default;
```
